Code for reproducing the results in [1].

Main files:
- examples.m reproduces the examples disseminated in the article.
- application1_iga.m computes an approximate Kronecker factorization of a real matrix using the Kronecker structure of its sparsity pattern. Note: this application requires Matlab's Tensor Toolbox [1].
- application2_networks.m visualizes the Kronecker graph of a network.

References:
[2] Y. Voet and L. De Novellis. Identifying Kronecker product factorizations, 2025.
[1] B. W. Bader and T. G. Kolda. Tensor toolbox for MATLAB, version 3.6, 2023.
